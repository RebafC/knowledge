# Awesome

A highly opinionated list of useful PHP tools:

* [assert: assertion library](https://github.com/beberlei/assert)
* [HTTPlug: PSR-7 client abstraction library](http://httplug.io/)
* [humbug: Mutation Testing framework](https://github.com/padraic/humbug#introduction)
* [iter: iteration primitive library](https://github.com/nikic/iter)
* [Monolog: PSR-3 log library](https://github.com/Seldaek/monolog/blob/master/doc/01-usage.md#using-monolog)
* [Phinx: Database Migrations tool](https://phinx.org/)
* [phpspec: SpecBDD framework](http://phpspec.net/)
* [Pomm: PostgreSQL library](http://www.pomm-project.org/)
* [porpaginas: pagination abstraction library](https://github.com/beberlei/porpaginas)
* [Tactician: Command Bus library](https://tactician.thephpleague.com/)
* [uuid: UUID generator library](https://benramsey.com/projects/ramsey-uuid/)
